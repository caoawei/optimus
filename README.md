
# optimus
简介: 个人日常所写java项目,各个子工程均可再未来为整个optimus项目服务
基础技术: spring-boot+spring-cloud+mybatis+mysql

### optimus-log
简介: 日志工程
功能: 收集,存储,检索,分析等
状态: 未开发


### optimus-reptile
简介: 爬虫相关
功能: 爬网站数据,分析等
状态: 未开发

### optimus-courier
简介: RPC工程
功能: 进程通讯(娱乐项目)
状态: 未开发


有兴趣者可对以上任意一个子工程进行开发

(其余的子工程均是业务相关)
